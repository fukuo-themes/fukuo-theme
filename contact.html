Installation

How do I install a theme?

Starting as of now, my themes will be uploaded on the Theme Garden which is easy to install. Just click the install button and you're done. As for installing a third-party theme, all you have to do is copy-paste the code I have provided in my theme preview. It will leads you to the pastebin/github.

How do I install a custom page?

To create a page with a custom layout:

  1. Click “Add a page” in the Pages section of your Customize menu.
  2. Select "Custom Layout" from the dropdown menu.
  3. Insert custom code into the HTML Editor.
  4. Type a page URL after the slash (/). For example, an about page could be /about and a contact page could be /contact.
  5. Add a title in the Page Title field.
  6. Enable "Show a link to this page." then click "Save"

Where's your old themes? I can't seem to find them on your blog...

I have decided to delete them because they are outdated and I don't have the want to update the theme. As time goes, my view of "Tumblr themes" changed and I don't want to provide the ugly design and not functional for you. Fortunately, Theme Hunter archived my old ones so if you still want to use them, go ahead (but I don't provide a support for them).

Can you please help me? I am using someone else's theme...

No. Please keep in mind that I do not help with other people's theme. Just ask the owner of the theme instead of asking me.

How did you learn to code?

I am entirely self-taught. Back then in 2011, I self-learned HTML and CSS when I can't find the Tumblr theme that suits my needs so I was playing around with the code and got familiar with it. In 2014-2015, I decided to take a course in Javascript and Python to improve my skills in programming language. Some of my resources are w3schools, CSS Tricks, Codeacademy and Tumblr theme documentation.
